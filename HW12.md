13. Find the solution to $a_n = 7a_{n-2} + 6a_{n-3}$ with $a_0 = 9$, $a_1 = 10$, and $a_2 = 32$.
### answer
Then 
$$
a_n -0*a_{n-1}-7*a_{n-2}-6*a_{n-3} = 0 \\
x^3-7x-6=0  \\
(x-3)(x-2)(x-1) = 0
$$
so $a_n = a * 3^n + b * 2^n + c * 1^n$
$a=10,b=-19,c=18$
